Mister Barista
by Karl Lilje

To Run:
Start index.html with :
npx serve

Object: 
Select a cup of coffee. 
If youe choice matches what the Barista poured, you win 1 coin. If you guess wrong, you lose 1 coin.

Uses a mock game server to simulate game logic
